# Labels

## Here you can find all labels used inside components.

You have to wrap your app in <code>{'<LabelProvider labels={labels} locale={locale}>'}</code> and pass
in your own labels.

However components fall back to translations defined in the library itself. Currently we have default
translations for Estonian, English and Russian. You can control what default translations are used with <code>locale</code> prop.
If <code>locale</code> prop is not set, then the default languages is English.
