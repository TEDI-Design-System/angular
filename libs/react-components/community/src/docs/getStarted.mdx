# Get started with TEHIK tedi-design-system

### Installation

```bash
npm install @tehik-ee/tedi-design-system
```

### Labels - LabelProvider

Tehik react-components use labels that are provided with <b>{'<LabelProvider>'}</b> component. Every
application should be wrapped with LabelProvider to provider all necessary labels for users (mostly for
accessibility). See all labels under [Documentation/Labels](/docs/documentation-labels--docs).
More documentation about LabelProvider can be found under [components](/docs/components-labelprovider--docs).

#### 1. Wrap Application with LabelProvider

```tsx
<LabelProvider labels={labels} locale={locale}>
  <YourApp />
</LabelProvider>
```

### Styles - StyleProvider

tedi-design-system are depending on some global styles. Follow next steps to import styles to your
application.

#### 1. Wrap Application with StyleProvider

```tsx
<StyleProvider>
  <YourApp />
</StyleProvider>
```

#### 2. Add styles import to the root of application

```tsx
import '@tehik-ee/tedi-design-system/index.css';
```
